Dice Rolling Game

A simple dice rolling game. The game is played with two players. The first die is assigned to player 1 and the second die is assigned to player 2. Whenever the 'roll' button is pressed the dice are rolled, and the player with the highest number wins. The h1 element is updated accordingly. Behind the scenes, the images of the dice are fetched and stored in an array. Random numbers are generated for both players and compared. After these numbers are obtained, they are used to retrieve the images from the array and update the DOM and display the winner.
Demo Link - Live Demo

Simon Game

A game based on the Simon Game. The game works in the following manner: the user has to press the buttons in the correct order. If the user presses the wrong button, the game ends. At which point the screen 'flashes' red and the game is over. Everytime the user presses the right sequence, a new sequence is generated, displayed to the user as a level. A random number is generated which is then assigned to a color, then this color is added to the game pattern, and then displayed to the user. The corresponding sound is also played.
Demo Link - Live Demo
