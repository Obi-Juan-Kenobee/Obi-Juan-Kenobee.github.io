A game with vanilla JavaScript

This project is a work in progess.

It is a game that will be a horizontal scrolling game (think of the old Mario Bros game in the 1980s).

This game will be built entirely with vanilla JavaScript, no frameworks of any kind will be used.
I will be using animations in canvas (which I am currently learning more in depth).
For now, you are able to only move the character with the arrow keys on the keyboard.

Note: This game displays on mobile devices just fine, however, it does not currently support mobile controls.
